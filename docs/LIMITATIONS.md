# LIMITATIONS

1. `#include` пока не реализован.
- При активной директиве `#include`:
- `enable_include = 0` -> `PP_STATUS_INCLUDE_DISABLED`
- `enable_include = 1` -> `PP_STATUS_INTERNAL_ERROR`

2. Функциональные макросы не поддерживаются.
- Поддерживается только объектная форма `#define NAME ...`.

3. Рекурсивная/многошаговая подстановка не выполняется.
- Подстановка делает один проход по активной строке.

4. Макросы не подставляются в директивах.
- Например, имена в `#ifdef` читаются как литеральные идентификаторы.

5. Обработка комментариев ограничена C-стилем.
- `// ...`
- `/* ... */` при `support_block_comments = 1`

6. Нет режима сохранения source-map.
- Диагностика возвращает позицию ошибки, но карта преобразований вход->выход не строится.
